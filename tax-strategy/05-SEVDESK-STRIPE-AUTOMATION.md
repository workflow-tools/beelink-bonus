# SevDesk + Stripe Automation Plan

---

## Why SevDesk

SevDesk is a German cloud accounting platform designed for small businesses and Einzelunternehmen/GmbH/UG. It handles:

- Invoice generation (Rechnungen) with proper German formatting
- Expense tracking and receipt capture
- Bank account reconciliation
- VAT (Umsatzsteuer) pre-returns (Voranmeldung)
- Annual financial statement preparation
- Direct ELSTER integration for tax filing
- Steuerberater DATEV export

For a German UG selling internationally, SevDesk is the natural accounting backbone.

---

## Integration Architecture

```
Stripe (payments)
    │
    ├── Webhook → Make.com (automation) → SevDesk (invoice creation)
    │
    └── Payout → Wise (bank) → SevDesk (bank reconciliation)

SevDesk
    │
    ├── → DATEV export → Steuerberater
    ├── → ELSTER → Finanzamt (VAT returns)
    └── → Reports → You (operating dashboard)
```

## Option A: Stripe App Marketplace — SevDesk by Wiggers

There is a SevDesk connector available in the Stripe App Marketplace ("sevDesk by Wiggers"). This provides a direct integration without needing Make.com.

### What It Does
- Automatically creates SevDesk invoices from Stripe payments
- Syncs customer data
- Maps Stripe products to SevDesk revenue categories

### Setup
1. Install "sevDesk by Wiggers" from Stripe App Marketplace
2. Authenticate with your SevDesk account
3. Map Stripe products (WritingPAD, ProPrecept IE, etc.) to SevDesk revenue accounts
4. Configure invoice templates (German formatting, Impressum, USt-IdNr)

### Limitations
- May not handle multi-currency elegantly
- Check if it supports separate VAT treatment per jurisdiction
- May need supplementing with Make.com for complex workflows

## Option B: Make.com Automation (More Flexible)

Make.com (formerly Integromat) provides visual workflow automation between Stripe and SevDesk with full customization.

### Workflow 1: New Stripe Payment → SevDesk Invoice

**Trigger:** Stripe webhook — `invoice.paid` event
**Steps:**
1. Extract customer name, email, amount, currency, product, VAT amount
2. Look up or create customer in SevDesk
3. Create invoice in SevDesk with:
   - Correct currency (GBP/EUR/ZAR/AUD/PHP)
   - Correct VAT rate for the customer's jurisdiction
   - Product line item matching Stripe product name
   - Payment status: "paid"
   - Invoice number: auto-generated by SevDesk
4. Attach to correct SevDesk revenue category (per product/market)

### Workflow 2: Stripe Refund → SevDesk Credit Note

**Trigger:** Stripe webhook — `charge.refunded` event
**Steps:**
1. Find the original SevDesk invoice
2. Create a Gutschrift (credit note) in SevDesk
3. Link to original invoice

### Workflow 3: Monthly Stripe Payout Reconciliation

**Trigger:** Scheduled — 1st of each month
**Steps:**
1. Pull Stripe payout report for prior month
2. Match against Wise bank statement import in SevDesk
3. Flag any discrepancies for manual review

### Make.com Pricing
- Free tier: 1,000 operations/month (sufficient for early stage)
- Core plan: $10.59/month for 10,000 operations
- Scale as transaction volume grows

## DATEV Export for Steuerberater

SevDesk supports DATEV export, which is the standard data format used by German Steuerberater. Monthly or quarterly:

1. SevDesk → Exporte → DATEV
2. Select period
3. Export includes: all invoices, expenses, bank transactions, VAT calculations
4. Send to Steuerberater or grant them direct SevDesk access (Steuerberater-Zugang)

**Recommendation:** Grant your Steuerberater direct read-only access to SevDesk. This eliminates manual export steps and lets them pull data whenever needed.

---

## ELSTER Integration for VAT Returns

SevDesk can submit Umsatzsteuervoranmeldungen (VAT advance returns) directly to the Finanzamt via ELSTER:

1. SevDesk calculates VAT collected and input VAT from your transactions
2. Generates the Voranmeldung
3. You review and approve
4. SevDesk submits via ELSTER

**Note:** This covers DOMESTIC German VAT only. UK VAT and EU OSS returns are filed separately.

## Automation Roadmap

### Phase 1: Foundation (Before First Revenue)
- [ ] Create SevDesk account (Buchhaltung plan, ~€17/month)
- [ ] Configure company details and invoice template
- [ ] Set up revenue and expense categories per market
- [ ] Connect Wise bank account to SevDesk
- [ ] Install "sevDesk by Wiggers" Stripe connector OR set up Make.com workflows
- [ ] Grant Steuerberater access

### Phase 2: First Revenue (UK + Ireland Launch)
- [ ] Verify: each Stripe payment auto-generates a SevDesk invoice
- [ ] Verify: correct VAT rates applied per jurisdiction
- [ ] Verify: Wise payouts reconcile against SevDesk records
- [ ] Run first monthly close: revenue, expenses, profit, tax reserves

### Phase 3: Scale (3+ Markets Active)
- [ ] Add SevDesk categories for new markets (RSA, AU, PH)
- [ ] Automate refund/credit note creation
- [ ] Build a monthly dashboard report (revenue by market, tax reserves, operating cash)
- [ ] Consider SevDesk API for custom reporting if needed

### Phase 4: GmbH Conversion
- [ ] SevDesk tracks accumulated GmbH reserve on balance sheet
- [ ] Steuerberater uses SevDesk data for conversion filing
- [ ] Update company details in SevDesk post-conversion
